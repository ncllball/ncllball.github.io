Maintainer: Early Season Ramp Visibility

Goal: After June 1 each year, hide early-season ramp highlight blocks (still leave data available for context). Current approach uses CSS class early-season-visible on April/May ramps. To hide:

  • Open Baseball Early Season Section & Softball counterpart (Softball Early Ramp)
  • Change section class from early-season-visible to early-season-hidden
  • Leave anchor id intact so legacy links still resolve
  • Optionally add a short note inside section noting it is now off-season (do not delete historical table)

Future Enhancement: Could automate with a tiny JS date check; intentionally avoided for static simplicity.
