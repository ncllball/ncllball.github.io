# Volunteer Lite — Draft Concept

> Draft for potential addition to helping-hub or volunteer-hub. Volunteer Lite is a no-commitment way to help the league — sign up for a list, get tapped when we need hands, say yes or no. No season-long obligation.

---

## Concept

We create targeted lists for one-off events. When we need help, we reach out to the list — if you can make it, great; if not, maybe next time. No pressure, no long-term commitment, no prior baseball knowledge required.

These lists are **in addition to** — not in place of — coaching, assistant coaching, and team manager roles.

## The Lists

Sign up for any or all:

- **Home Run Derby** — help run our annual HRD event
- **Photo Day** — help wrangle players, assist the photographer
- **Field Work** — help prep or clean up fields on tough-weather days
- **Coaches Meeting** — setup and cleanup for the annual coaches meeting

## Talent List

Have a skill that could help the league throughout the year? We want to hear from you. Examples:
- Photography
- Graphic design
- Handyperson / field maintenance
- Other — tell us what you've got

---

Anyone can sign up. Whatever you can offer, we can use it.
