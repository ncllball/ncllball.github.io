Footnote & Note Patterns

Demo page — all 4 patterns using the live site stylesheet. Not a CMS page.

Pattern 1 section.footies + sup.footie-ref — non-collapsible numbered refs (most common)

Use this pattern when you have 2–4 short notes following a list or table.

  • Players must be league age 10–12 by August 31, 2026.1
  • Tryout attendance is required unless a medical waiver is on file.2
  • Travel costs are the responsibility of the family.3

1 League age is calculated per Little League's official birth-date chart.

2 Waivers must be submitted to the Player Agent no later than 24 hours before tryouts.

3 Financial Aid is available — see the Financial Aid page for details.

Pattern 1 variant section.footies.footies_red — same but red top border

Add .footies_red to the section to use a red top border instead of light gray.

  • All-Star selection follows Little League Rules and NCLL Local Rules.1

1 Selection is consensus-based and conducted to preserve confidentiality.

Pattern 2 details.footies — collapsible, good for long explanatory notes

Use when notes would clutter the page — reader can expand only if needed.

Division League Ages Season
Majors 10–12 Spring/Summer
AAA 9–111 Spring only
Juniors 13–14 Spring/Summer
Division Notes

1AAA Placement Players league age 9 may play AAA if evaluated ready by the Player Agent and manager. This is an exception, not the default.

2Juniors Eligibility A player who turns 15 before May 1 is not eligible for the Juniors division and must play Seniors.

Pattern 3 section.footies + .info-icon — single standalone notice, no numbered ref

Use for one-off notices not tied to a numbered reference in the content above.

Registration fees are listed below. Uniforms are included.

Scholarship assistance is available. Apply at the Financial Aid page — no one is turned away for financial reasons.

Pattern 4 p.note inside .pitching-ref — reference/table context only

Scoped to .pitching-ref .note in CSS. Do not use outside pitching-reference pages.

*Rest requirements are based on pitch count thresholds, not just day count. Consult the current Little League rulebook for the full rest table.
